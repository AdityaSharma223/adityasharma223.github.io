Vim plugins and plugin managers.

(back)

VIM plugings are used to make your vim feel more comfortable, these include the airline (the mode display at the bottom) or auto complete something you may find in more modern or normie friendly editors (btw vim does have a built in autocomplete feature, try Ctrl-N :p). These plugins can be installed with the help of plugin managers, there are a whole bunch of them but the one that I use is Vim Plug it is fast and easy to use. You can install it with a few commands.
